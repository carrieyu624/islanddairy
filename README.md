## 📖 简介

这是「岛屿日记」的源码仓库。  
它是一个单文件的静态 HTML 网站，采用青色调主题，按时间倒序排列日记条目，每条日记都带有精确的时间戳。

**初衷**：  
让文字回归最原始的状态——就像最早写博客那样，只为自己记录、与自己对话。不想被社交媒体的「观众视角」绑架，这里没有点赞、没有流量、只有纯粹的文字和真实的思考。
